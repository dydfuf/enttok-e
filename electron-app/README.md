# Enttok-e Electron App

Local-first work journal with AI-powered suggestions.

## Requirements
- Node.js and pnpm
- Python + uv (for the backend in `../backend`)

## Development
```bash
pnpm install
pnpm dev
```

## Build
```bash
pnpm build
```

## Package
```bash
pnpm package
```

## Lint / Format
```bash
pnpm lint
pnpm format
pnpm check
```
